# Simulador de Préstamos

Este proyecto es un simulador de préstamos desarrollado en JavaScript con Node.js. Permite calcular la cuota mensual de un préstamo, el total a pagar, los intereses generados y una tabla básica de amortización.

## Descripción

El programa solicita al usuario los siguientes datos:

* Nombre
* Monto del préstamo
* Plazo en meses

Con esta información, el sistema calcula la cuota mensual usando una tasa de interés mensual fija del **1.97%**. Además, muestra una tabla de amortización con el saldo inicial, interés mensual, abono a capital, cuota y saldo final de cada mes.

## Tecnologías utilizadas

* JavaScript
* Node.js
* Librería readline-sync

## Instalación

Para ejecutar el proyecto, primero clona el repositorio:

git clone URL_DEL_REPOSITORIO

Luego ingresa a la carpeta del proyecto:

cd NOMBRE_DEL_REPOSITORIO

Instala la dependencia necesaria:

npm install 

## Ejecución

Ejecuta el archivo principal con Node.js:

node SimuladorPrestamo.js

## Ejemplo de uso

El programa solicitará los datos por consola:

Ingrese su nombre: Manuel
Ingrese el monto del prestamo: 1000000
Ingrese el plazo en meses: 12

Luego mostrará el resumen del préstamo y la tabla de amortización correspondiente.
